1.Ansible installation

MASTER WORK

$ sudo apt-get update
$ sudo apt install software-properties-common
$ sudo apt-add-repository ppa:ansible/ansible
$ sudo apt update
$ sudo apt-get install ansible

NODE/SLAVE WORK

$ sudo apt-get update
$ sudo apt install python3
